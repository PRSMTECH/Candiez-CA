# Candiez Quick Reference Card
## Daily Operations Cheat Sheet

Print this card and keep it at each register for quick reference.

---

## Starting Your Day

```
1. Log in at candiez.shop
2. Go to POS
3. Count your cash drawer
4. Enter starting amount -> Start Shift
```

---

## Processing a Sale

| Step | Action |
|------|--------|
| 1 | Search/browse products |
| 2 | Click to add to cart |
| 3 | Adjust quantities with +/- |
| 4 | Click "Select Customer" (ask: "Are you a member?") |
| 5 | Apply loyalty points if desired |
| 6 | Click Checkout |
| 7 | Select Cash or Debit |
| 8 | Complete sale! |

---

## Customer Lookup

**In POS:**
- Click "Select Customer"
- Search by name, phone, or email

**In Customers page:**
- Use search bar at top
- Browse alphabetically

---

## Loyalty Points

| Tier | Points Needed | Multiplier |
|------|---------------|------------|
| Bronze | 0 | 1x |
| Silver | 500 | 1.25x |
| Gold | 1,500 | 1.5x |
| Platinum | 5,000 | 2x |

**Earning:** $1 = 1 point (multiplied by tier)
**Redeeming:** 100 points = $1 discount

---

## New Customer Registration

```
Customers -> + New Customer -> Fill form -> Save
Required: First name, Last name, Phone, DOB
```

---

## Stock Adjustment (Manager+)

```
Inventory -> Find product -> Adjust Stock
- Enter quantity (+add, -remove)
- Select reason
- Add notes
- Confirm
```

---

## Void vs Refund (Manager+)

| Use | When | Result |
|-----|------|--------|
| **VOID** | Entire sale was a mistake | Cancels whole transaction |
| **REFUND** | Customer returns item(s) | Returns specific items |

Both restore inventory automatically.

---

## Common Issues

| Problem | Solution |
|---------|----------|
| Can't log in | Check spelling, contact admin |
| POS stuck | Refresh page (Ctrl+R) |
| Stock wrong | Make adjustment with reason |
| Points missing | Check customer was linked to sale |
